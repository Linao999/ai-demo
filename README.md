# 🤖 AI 产品能力演示

**两个 AI 应用场景的交互式 Demo**

## Tab 1: 智能客服 💬

模拟电商场景下的 AI 客服系统，展示：
- **RAG 知识库检索**：根据售后政策、产品库自动匹配答案
- **意图识别**：自动识别用户是退货/咨询/投诉
- **情绪感知**：实时分析用户情绪，触发安抚或转人工
- **多轮对话**：上下文关联，不丢失对话脉络

## Tab 2: 智能旅游规划 🗺️

基于 LLM 的个性化旅行规划引擎，展示：
- **多维度匹配**：目的地 × 天数 × 预算 × 风格
- **结构化输出**：每日行程 + 预算明细 + 实用贴士
- **可执行性**：精确到时段、交通方式、费用预估

## 设计理念

这不是"炫技"demo，重点展示 AI 产品经理的核心能力：
1. **问题定义**：为什么这个场景需要 AI 而非传统方案
2. **交互设计**：如何在 UI 中透明展示 AI 的推理过程
3. **可控性**：如何处理 AI 的不确定性（置信度标注、转人工机制）
4. **评测思维**：每个回复附带推理过程，方便评估质量
